# Diabetes Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a patient is diabetic or non-diabetic using machine learning techniques based on health parameters.

## 🎯 Objective
To build a predictive model that assists in early detection of diabetes and supports data-driven healthcare decisions.

## 📊 Dataset
- Pima Indians Diabetes Dataset
- Features: Glucose, BMI, Age, Blood Pressure, etc.

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## 🧠 Model
- Logistic Regression
- Train-Test Split (80/20)

## 📈 Results
- Accuracy: ~76%
- Precision: 67%
- Recall: 63%
- F1 Score: 65%

## 🔍 Key Insights
- Glucose is the most important feature
- BMI and Age significantly influence prediction
- Some diabetic cases are missed → improvement needed

- Improve model accuracy
- Integrate real-time healthcare data
- Build web application
